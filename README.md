# Flutter Workshop - 2 Day Course

[![Flutter](https://img.shields.io/badge/Flutter-Framework-blue?logo=flutter)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-Language-0175C2?logo=dart)](https://dart.dev)

## 📚 คำอธิบายคอร์ส (Course Description)

คอร์สอบรมเชิงปฏิบัติการ Flutter เข้มข้น 2 วัน สำหรับผู้ที่ต้องการเริ่มต้นพัฒนาแอปพลิเคชันมือถือด้วย Flutter Framework จากพื้นฐานจนถึงการสร้างแอปพลิเคชันที่สมบูรณ์

This intensive 2-day Flutter workshop is designed for beginners who want to start developing mobile applications using Flutter Framework, from basics to building complete applications.

## 🎯 วัตถุประสงค์ (Objectives)

เมื่อจบคอร์สนี้ ผู้เข้าอบรมจะสามารถ:

### ทักษะด้านเทคนิค (Technical Skills)
- ✅ **Flutter Fundamentals**: เข้าใจ Flutter framework, architecture, และ widget tree
- ✅ **Dart Programming**: เขียน Dart code ได้อย่างมั่นใจ รวมถึง OOP concepts และ null safety
- ✅ **UI Development**: สร้างและออกแบบ User Interface ที่สวยงามด้วย Flutter widgets
- ✅ **State Management**: จัดการ state ในแอปพลิเคชันด้วย setState และ Provider
- ✅ **Navigation**: สร้าง multi-screen applications พร้อม navigation flows
- ✅ **API Integration**: เชื่อมต่อกับ REST APIs และจัดการ JSON data
- ✅ **Data Persistence**: บันทึกและโหลดข้อมูลจาก local storage
- ✅ **Testing**: เขียน unit tests และ widget tests
- ✅ **Deployment**: Build และ deploy แอปพลิเคชันบน Android และ iOS

### ทักษะปฏิบัติ (Practical Skills)
- 🔨 **Project Building**: สร้างแอปพลิเคชันที่สมบูรณ์ตั้งแต่ต้นจนจบ
- 🎨 **UI/UX Design**: ออกแบบ interface ที่ใช้งานง่ายและสวยงาม
- 🐛 **Debugging**: แก้ไขปัญหาและ debug Flutter applications
- 📚 **Best Practices**: ใช้ coding standards และ best practices ของ Flutter community
- 🚀 **Continuous Learning**: มีทักษะในการเรียนรู้และค้นคว้าต่อยอดด้วยตนเอง

### ผลลัพธ์การเรียนรู้ (Learning Outcomes)
หลังจากจบคอร์ส คุณจะสามารถ:
1. **วันที่ 1**: สร้าง Flutter app แรกที่มี interactive UI
2. **วันที่ 2**: พัฒนา full-featured mobile application ที่เชื่อมต่อกับ API
3. **หลังคอร์ส**: พัฒนาแอปพลิเคชันของคุณเองได้อย่างอิสระ

## 📋 ข้อกำหนดเบื้องต้น (Prerequisites)

### ความรู้พื้นฐาน (Required Knowledge)
- **การเขียนโปรแกรม**: ความเข้าใจพื้นฐานเกี่ยวกับ variables, functions, loops, และ conditions ในภาษาใดก็ได้
- **Object-Oriented Programming**: ความเข้าใจเบื้องต้นเกี่ยวกับ classes และ objects (เป็นประโยชน์)
- **Mobile App Concepts**: ความคุ้นเคยกับ mobile applications เบื้องต้น

- **RAM**: อย่างน้อย 8GB (แนะนำ 16GB สำหรับประสบการณ์ที่ดีที่สุด)

### 1. ติดตั้ง Flutter SDK (Step-by-Step)

#### macOS

**Option 1: Using Homebrew (แนะนำ)**
```bash
# ติดตั้ง Homebrew ถ้ายังไม่มี
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# ติดตั้ง Flutter
brew install flutter

# ตรวจสอบการติดตั้ง
flutter --version
```

**Option 2: Manual Installation**
```bash
# 1. ดาวน์โหลด Flutter SDK
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable

# 2. เพิ่ม Flutter ใน PATH
export PATH="$PATH:$HOME/development/flutter/bin"

# 3. เพิ่มใน shell profile เพื่อให้ถาวร
echo 'export PATH="$PATH:$HOME/development/flutter/bin"' >> ~/.zshrc

# 4. Reload shell configuration
source ~/.zshrc

# 5. ตรวจสอบการติดตั้ง
flutter --version
```

#### Windows

**ขั้นตอนการติดตั้ง:**
1. **ดาวน์โหลด Flutter SDK**
   - ไปที่: https://docs.flutter.dev/get-started/install/windows
   - ดาวน์โหลด Flutter SDK zip file

2. **Extract ไฟล์**
   - แตกไฟล์ไปยัง `C:\src\flutter` (หรือตำแหน่งที่ต้องการ)
   - ⚠️ หลีกเลี่ยงโฟลเดอร์ที่มีเครื่องหมายพิเศษหรือช่องว่าง

3. **เพิ่ม Flutter ใน PATH**
   - กด Windows key และค้นหา "Environment Variables"
   - คลิก "Edit the system environment variables"
   - คลิก "Environment Variables..."
   - ใน "User variables" หาตัวแปร "Path"
   - คลิก "Edit" → "New"
   - เพิ่ม `C:\src\flutter\bin`
   - คลิก "OK" ทั้งหมด

4. **เปิด Command Prompt ใหม่และทดสอบ**
   ```cmd
   flutter --version
   ```

#### Linux (Ubuntu/Debian)

```bash
# 1. ติดตั้ง dependencies ที่จำเป็น
sudo apt-get update
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa

# 2. Clone Flutter repository
cd ~/development
git clone https://github.com/flutter/flutter.git -b stable

# 3. เพิ่ม Flutter ใน PATH
export PATH="$PATH:$HOME/development/flutter/bin"

# 4. เพิ่มใน .bashrc เพื่อให้ถาวร
echo 'export PATH="$PATH:$HOME/development/flutter/bin"' >> ~/.bashrc

# 5. Reload configuration
source ~/.bashrc

# 6. ตรวจสอบการติดตั้ง
flutter --version
```

### 3. ติดตั้ง IDE และ Extensions

#### VS Code (แนะนำสำหรับผู้เริ่มต้น)

**ขั้นตอนการติดตั้ง:**
1. ดาวน์โหลดจาก: https://code.visualstudio.com/
2. ติดตั้ง VS Code
3. เปิด Extensions (กด `Ctrl+Shift+X` หรือ `Cmd+Shift+X`)
4. ค้นหาและติดตั้ง:
   - ✅ **Flutter** (by Dart Code) - รวม Dart extension แล้ว
   - ✅ **Awesome Flutter Snippets** (optional - code snippets)
   - ✅ **Dart Data Class Generator** (optional - utility)

**การตรวจสอบ:**
- กด `Ctrl+Shift+P` (หรือ `Cmd+Shift+P`)
- พิมพ์ "Flutter"
- ควรเห็นคำสั่ง Flutter ต่างๆ

#### Android Studio (แนะนำสำหรับผู้ที่มีประสบการณ์)

**ขั้นตอนการติดตั้ง:**
1. ดาวน์โหลดจาก: https://developer.android.com/studio
2. ติดตั้ง Android Studio
3. เปิด Android Studio
4. ไปที่:
   - **Windows/Linux**: File → Settings → Plugins
   - **macOS**: Android Studio → Preferences → Plugins
5. ค้นหาและติดตั้ง:
   - ✅ **Flutter** plugin
   - ✅ **Dart** plugin (จะถูกติดตั้งอัตโนมัติกับ Flutter)
6. Restart Android Studio

### 4. ตั้งค่า Android Emulator

**ขั้นตอนสำหรับ Android Studio:**
1. เปิด Android Studio
2. ไปที่ **Tools → Device Manager** (หรือ **AVD Manager**)
3. คลิก **"Create Device"**
4. เลือก hardware profile (แนะนำ: Pixel 5 หรือ Pixel 7)
5. คลิก **"Next"**
6. เลือก system image:
   - แนะนำ: **API Level 33** (Android 13) หรือใหม่กว่า
   - คลิก **"Download"** ถ้ายังไม่ได้ดาวน์โหลด
7. คลิก **"Next"** → **"Finish"**

**ทดสอบ Emulator:**
```bash
# ดูรายการ emulators ที่มี
flutter emulators

# เปิด emulator
flutter emulators --launch <emulator_id>

# ตรวจสอบ devices ที่เชื่อมต่อ
flutter devices
```

### 5. ตั้งค่า iOS Simulator (macOS เท่านั้น)

**ขั้นตอนการติดตั้ง:**
1. ติดตั้ง Xcode จาก App Store
2. เปิด Xcode และยอมรับ license agreement
3. ติดตั้ง Command Line Tools:
   ```bash
   sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
   sudo xcodebuild -runFirstLaunch
   ```

**เปิด Simulator:**
```bash
# เปิด iOS Simulator
open -a Simulator

# หรือใช้ผ่าน Xcode
# Xcode → Open Developer Tool → Simulator
```

**ตรวจสอบ devices:**
```bash
flutter devices
```

### 6. ตั้งค่า Chrome สำหรับ Web Development

**ตรวจสอบการติดตั้ง:**
```bash
# ตรวจสอบว่า Chrome พร้อมใช้งาน
flutter devices
```

**เปิดใช้งาน Web:**
```bash
# Enable web support (ถ้ายังไม่ได้เปิด)
flutter config --enable-web

# ตรวจสอบการตั้งค่า
flutter config
```

## 🔧 Troubleshooting Guide

### Common Issues และวิธีแก้ไข

#### 1. Flutter Doctor Issues

**Problem:** ❌ Android toolchain not found
```bash
# Solution:
flutter doctor --android-licenses
# Accept all licenses by typing 'y'
```

**Problem:** ❌ Android Studio not found
```bash
# Solution:
# 1. Download Android Studio from https://developer.android.com/studio
# 2. Install Android Studio
# 3. Run flutter doctor again
```

**Problem:** ❌ Command line tools not available
```bash
# Solution:
# Open Android Studio → Preferences → Appearance & Behavior 
# → System Settings → Android SDK → SDK Tools
# Check "Android SDK Command-line Tools" → Apply
```

#### 2. Build และ Run Issues

**Problem:** Build fails with "Gradle task assembleDebug failed"
```bash
# Solution 1: Clean and rebuild
flutter clean
flutter pub get
flutter run

# Solution 2: Check internet connection
# Gradle needs to download dependencies

# Solution 3: Update Gradle wrapper
cd android
./gradlew wrapper --gradle-version=8.0
cd ..
```

**Problem:** "Waiting for another flutter command to release the startup lock"
```bash
# Solution: Remove the lock file
rm -rf ~/.flutter-devtools/.pid
# or
rm -rf $FLUTTER_ROOT/bin/cache/lockfile
```

**Problem:** Hot reload not working
```bash
# Solution 1: Try hot restart (press 'R')
# Solution 2: Stop and restart app completely
# Solution 3: Clean project
flutter clean
flutter pub get
flutter run
```

#### 3. Emulator/Device Issues

**Problem:** No devices found
```bash
# Check devices
flutter devices

# For Android Emulator:
flutter emulators
flutter emulators --launch <emulator_id>

# For iOS Simulator (macOS):
open -a Simulator

# For physical device:
# - Enable Developer Mode
# - Enable USB Debugging
# - Accept computer authorization
```

**Problem:** Emulator very slow
```bash
# Solution 1: Allocate more RAM
# Android Studio → AVD Manager → Edit → Advanced Settings → RAM

# Solution 2: Enable Hardware Acceleration
# Settings → Enable Intel VT-x or AMD-V in BIOS

# Solution 3: Use a simpler device
# Create emulator with lower API level (API 28-30)
```

#### 4. Dependencies Issues

**Problem:** "Package not found" error
```bash
# Solution:
flutter clean
rm pubspec.lock
flutter pub get

# If still fails:
flutter pub cache repair
```

**Problem:** Version conflict errors
```bash
# Solution:
# Check pubspec.yaml for dependency conflicts
# Run:
flutter pub upgrade
flutter pub get
```

#### 5. IDE Issues

**Problem:** VS Code not recognizing Flutter
```bash
# Solution:
# 1. Install Flutter extension
# 2. Reload VS Code (Cmd/Ctrl + Shift + P → "Reload Window")
# 3. Check Flutter SDK path (Cmd/Ctrl + Shift + P → "Flutter: Change SDK")
```

**Problem:** Android Studio Flutter plugin issues
```bash
# Solution:
# 1. File → Settings → Plugins
# 2. Uninstall Flutter and Dart plugins
# 3. Restart Android Studio
# 4. Reinstall plugins
# 5. Restart again
```

#### 6. Platform-Specific Issues

**macOS - Xcode Issues:**
```bash
# Problem: Xcode version too old
# Solution: Update Xcode from App Store

# Problem: CocoaPods not found
sudo gem install cocoapods
pod setup

# Problem: Signing issues
# Solution: Open Xcode → Project → Signing & Capabilities
# Select your development team
```

**Windows - Path Issues:**
```bash
# Problem: Flutter command not found
# Solution: Add Flutter to PATH
# 1. Search "Environment Variables"
# 2. Edit PATH variable
# 3. Add C:\src\flutter\bin (or your Flutter location)
# 4. Restart Command Prompt
```

**Linux - Permission Issues:**
```bash
# Problem: Permission denied
# Solution:
chmod +x $FLUTTER_ROOT/bin/flutter
chmod +x $FLUTTER_ROOT/bin/dart

# Problem: USB device not recognized
# Solution:
sudo usermod -aG plugdev $USER
# Then logout and login again
```

#### 7. Performance Issues

**Problem:** App runs slowly
```bash
# Solution 1: Use release mode
flutter run --release

# Solution 2: Profile the app
flutter run --profile
# Then use DevTools to identify bottlenecks

# Solution 3: Check device specifications
flutter devices -v
```

**Problem:** App size too large
```bash
# Solution: Build with optimization
flutter build apk --split-per-abi
flutter build appbundle

# Check what's taking space:
flutter build apk --analyze-size
```

#### 8. Error Messages Reference

**"A RenderFlex overflowed"**
```dart
// Problem: Widget doesn't fit in available space
// Solution: Wrap with Expanded or Flexible
Expanded(
  child: YourWidget(),
)
```

**"setState() called after dispose()"**
```dart
// Problem: Calling setState on disposed widget
// Solution: Check if mounted before setState
if (mounted) {
  setState(() {
    // your state changes
  });
}
```

**"Vertical viewport was given unbounded height"**
```dart
// Problem: ListView inside Column without height constraint
// Solution: Wrap ListView with Expanded
Column(
  children: [
    Expanded(
      child: ListView(...),
    ),
  ],
)
```
